# Pose-Detection

## Pendahuluan
Pose-Detection adalah proyek yang bertujuan untuk melakukan estimasi pose manusia dari video. Estimasi pose manusia memiliki peran penting dalam berbagai aplikasi, seperti mengukur latihan fisik, pengenalan bahasa isyarat, dan kontrol gerakan tubuh secara penuh. Misalnya, aplikasi ini bisa digunakan untuk yoga, tari, dan aplikasi kebugaran. Selain itu, deteksi pose juga bisa digunakan untuk menampilkan konten digital di dunia nyata dalam augmented reality.

Proyek ini menggunakan **MediaPipe Pose**, sebuah solusi machine learning untuk pelacakan pose tubuh yang presisi tinggi. MediaPipe Pose dapat mengidentifikasi 33 titik landmark 3D di seluruh tubuh dari video RGB. Teknologi ini berbasis **BlazePose**, yang juga mendukung **ML Kit Pose Detection API**.

## Isi Proyek
Proyek ini mencakup beberapa folder utama:
- **Media-Demo**: Contoh video demonstrasi pelacakan pose.
- **PoseVideos**: Kumpulan video pose yang digunakan untuk pengujian.
- **Source**: Sumber kode utama untuk implementasi pose detection.

### File Utama
- **LICENSE**: Proyek ini dilisensikan di bawah lisensi MIT.
- **requirements.txt**: Berisi daftar dependensi Python yang diperlukan untuk menjalankan proyek ini.
- **README.md**: Berisi dokumentasi dan deskripsi proyek.

## Kesimpulan
Proyek pose detection ini berhasil mengimplementasikan solusi estimasi pose manusia yang dapat beroperasi secara real-time di perangkat modern, baik di desktop maupun mobile. Dengan menggunakan MediaPipe Pose dan BlazePose, proyek ini menawarkan pelacakan pose dengan akurasi tinggi, yang dapat diadaptasi untuk berbagai aplikasi seperti kebugaran, augmented reality, dan lainnya. Proyek ini menunjukkan bahwa machine learning dapat digunakan secara efisien untuk mengidentifikasi gerakan tubuh dari video RGB dan mendukung pengalaman interaktif yang lebih kaya.
